# a-claymorphism-demo

A lightweight prototype that showcases the core traits of the claymorphism design trend:

- Pastel gradients that make surfaces feel soft and sculpted
- Long, diffused shadows that create a hovering effect
- Inner highlights and subtle noise for tactile realism

Open `index.html` in a browser to explore the layout.

## Styleguide
